## Objection handling

- **"€867 is a lot."** Compared to what? It replaces €580–1,850 of separate
  tools plus the integration work. Per person on a 20+ team it's under €45/month
  for the entire stack. The honest framing is that Enterprise is usually the
  *cheaper* option once you count everything it replaces.
- **"We only need the shop / only the CMS today."** Start there on Pro — but the
  moment a second need appears (booking, support chat, SSO, AI), you're either
  buying another tool or upgrading. Most teams find they cross that line within a
  quarter, and the trial lets them confirm the full value before committing.
- **"Switching is risky."** Enterprise includes white-glove onboarding, data
  migration from your current tools, and a dedicated success manager who runs the
  cutover during the 14-day trial — so you're live before the first invoice and
  never flying blind.
- **"What if we outgrow it?"** You don't — Enterprise is unlimited. Seats,
  workspaces, brands, and usage all scale with no plan change and no overage.
- **"Can we trust the uptime?"** Enterprise carries a contractual 99.9% SLA with
  priority response and service credits, plus a public status page. It's a
  commitment, not a promise.
